# Kebijakan Privasi ArusKas

Tanggal berlaku: 27 April 2026

ArusKas ("Aplikasi") menghargai privasi Anda. Kebijakan Privasi ini menjelaskan data apa yang diproses saat Anda menggunakan Aplikasi.

## 1. Ringkasan

- ArusKas tidak mewajibkan pendaftaran akun.
- Data keuangan yang Anda input diproses dan disimpan secara lokal di perangkat Anda.
- ArusKas tidak menjual data pribadi Anda.

## 2. Data yang Diproses

Aplikasi dapat memproses data berikut yang Anda masukkan sendiri:

- Nama profil (misalnya nama lengkap)
- Data transaksi keuangan (pemasukan, pengeluaran, kategori, tanggal, nominal)
- Pengaturan aplikasi (tema, ukuran font, preferensi terkait penggunaan aplikasi)

## 3. Cara Data Digunakan

Data digunakan untuk:

- Menampilkan ringkasan keuangan dan riwayat transaksi
- Menghasilkan laporan PDF bulanan
- Menyimpan preferensi tampilan dan kenyamanan penggunaan

## 4. Penyimpanan Data

- Data disimpan secara lokal pada perangkat Anda (penyimpanan aplikasi/peramban lokal).
- Data tidak dipindahkan ke server ArusKas selama penggunaan normal aplikasi.

## 5. Berbagi Data oleh Pengguna

Aplikasi menyediakan fitur ekspor dan berbagi laporan PDF.

- Berbagi hanya terjadi jika Anda secara aktif memilih tindakan berbagi.
- Tujuan berbagi bergantung pada aplikasi/layanan yang Anda pilih sendiri (misalnya aplikasi chat/email/drive).

## 6. Izin Aplikasi

Aplikasi menggunakan izin internet untuk kebutuhan operasional komponen aplikasi.

## 7. Keamanan Data

Kami berupaya menerapkan praktik wajar untuk melindungi data. Namun, tidak ada metode penyimpanan atau transmisi yang 100% aman.

## 8. Retensi dan Penghapusan Data

Karena data utama disimpan lokal, Anda dapat menghapus data dengan:

- Menghapus transaksi/profil dari dalam aplikasi (jika tersedia), atau
- Menghapus data aplikasi melalui pengaturan perangkat, atau
- Menghapus instalasi aplikasi.

## 9. Privasi Anak

Aplikasi tidak ditujukan khusus untuk anak di bawah usia yang diwajibkan hukum setempat. Jika Anda adalah orang tua/wali dan meyakini anak memberikan data pribadi, hubungi kami untuk tindak lanjut.

## 10. Perubahan Kebijakan Privasi

Kebijakan Privasi ini dapat diperbarui dari waktu ke waktu. Versi terbaru akan tersedia pada halaman kebijakan ini beserta tanggal berlaku terbaru.

## 11. Kontak

Untuk pertanyaan terkait privasi, hubungi:

- Email: support@aruskas.app
